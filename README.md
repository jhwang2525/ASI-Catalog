# ASI Catalog

A digital product catalog created for the ASI Show, designed to showcase products in an easy-to-browse online format. The catalog provides visitors with product information, imagery, and quick access through QR code integration.

## Features

* Responsive design for desktop and mobile devices
* Product showcase with images and descriptions
* Fast-loading static website
* QR code compatible for trade show distribution
* Easy to update and maintain

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/jhwang2525/ASI-Catalog.git
```

Navigate to the project folder:

```bash
cd ASI-Catalog
```

Open `index.html` in your browser or run a local development server.

## Project Structure

```text
ASI-Catalog/
├── index.html
├── css/
├── js/
├── images/
└── README.md
```

## Usage

This catalog is intended for product presentation and promotional use at industry events, including the ASI Show. Visitors can access the catalog directly through a web link or QR code.
